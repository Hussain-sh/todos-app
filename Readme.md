# Todo App

## Getting Started

### 1. Clone the repository

git clone https://github.com/Hussain-sh/todos-app.git

The repo contains two folders: `frontend/` and `backend/`

### 2. Frontend Setup

cd frontend
npm install

Create a `.env.local` file in the frontend folder:
NEXT_PUBLIC_API_URL=https://todos-app-production-2eae.up.railway.app/api

Run the development server:
npm run dev

Open http://localhost:3000 in your browser.

## Backend

The backend is already deployed and running on Railway.
No local setup required.
